# Handbook Feedback 说明

学习过程中如发现以下问题，请在此目录下创建 `.md` 文件记录：

- 错别字 / 语法错误
- 概念解释不清楚或有歧义
- 资料链接过期或不准确
- 结构建议（某章节应拆分、合并、调整顺序等）
- 缺乏某一主题的覆盖
- 有任何你在学习中的卡点和困惑

## 文件命名格式

```
YYYY-MM-DD-简短描述.md
```

示例：`2026-06-26-llm-token-count-example.md`

## 每条反馈的内容模板

```markdown
# 反馈标题

- **Handbook 页面**：[链接](https://aiweb3.school/zh/handbook/...)
- **问题描述**：
- **建议改法**：
- **来源**：（你是在什么场景下发现这个问题的）
```

## 反馈流向

你的 feedback 会累积在此目录。未来可以：

1. 定期整理，向 Handbook 仓库提交 Issue 或 Pull Request
2. 在共学社群中讨论
3. 作为你学习过程的原始材料索引
