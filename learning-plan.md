# 学习计划

初始化日期：2026-05-19
最近更新：2026-06-26

## 画像摘要

AI 新手 · Web3 有基础 · 无代码 → 目标开发 · 每日 ≤30 分钟

## 学习路径（三阶段）

### Phase 1：基础认知（第 1-3 周）

目标：建立 AI 概念框架，了解编程基本逻辑。

| 周次 | 主题 | Handbook 章节 | 每日行动 |
|------|------|---------------|----------|
| 第 1 周 | LLM 入门 | [AI/LLM](https://aiweb3.school/zh/handbook/ai/llm/) | 每天 1 个小节，记关键词 + 疑问 |
| 第 2 周 | Prompt 与 Context | [AI/Prompt](https://aiweb3.school/zh/handbook/ai/prompt/) + [AI/Context](https://aiweb3.school/zh/handbook/ai/context/) | 动手写 Prompt，观察效果 |
| 第 3 周 | 编程入门 | Python 或 JS 基础教程 | 每天 15 分钟语法，15 分钟 Handbook |

### Phase 2：AI + Web3 深入（第 4-6 周）

目标：理解 RAG、Agent、智能合约基础，开始写简单代码。

| 周次 | 主题 | Handbook 章节 | 备注 |
|------|------|---------------|------|
| 第 4 周 | RAG 基础 | [AI/RAG](https://aiweb3.school/zh/handbook/ai/rag/) | 概念为主，了解检索流程 |
| 第 5 周 | Agent 入门 | [AI/Agent](https://aiweb3.school/zh/handbook/ai/agent/) | Agent 概念、工具调用 |
| 第 6 周 | 智能合约入门 | [Web3/Smart Contract](https://aiweb3.school/zh/handbook/web3/smart-contract/) | 概念 + 读简单合约代码 |

### Phase 3：Bridge 与动手实践（第 7-10 周）

目标：理解 AI × Web3 交汇点，开始小项目。

| 周次 | 主题 | Handbook 章节 |
|------|------|---------------|
| 第 7 周 | Chain-aware Context | [Bridge/Chain-aware Context](https://aiweb3.school/zh/handbook/bridge/chain-aware-context/) |
| 第 8 周 | Agent Wallet | [Bridge/Agent Wallet](https://aiweb3.school/zh/handbook/bridge/agent-wallet/) |
| 第 9 周 | Web3 Tool Use | [Bridge/Web3 Tool Use](https://aiweb3.school/zh/handbook/bridge/web3-tool-use/) |
| 第 10 周 | 小项目 | 选一个 Track 做最小原型 |

## 打卡目标

- 每天一次打卡（≤5 分钟）+ 学习笔记（25 分钟）
- 每周末复盘一周内容
- 完成课程任务后提交到 tasks/

## 里程碑

- [ ] 读完 LLM 章节 — 第 1 周
- [ ] 写出第一个有效 Prompt — 第 2 周
- [ ] 理解 Agent 的基本运行方式 — 第 5 周
- [ ] 读懂一份简单 Solidity 合约 — 第 6 周
- [ ] 完成一个 AI × Web3 小原型 — 第 10 周

## Handbook 反馈

学习过程中如发现 Handbook 的错别字、概念不清晰、资料过期或结构建议，请一律记录到 `handbook-feedback/` 目录下。
